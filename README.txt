== Description ==

Simple plugin that let's you use Elementor forms to register visitors to your Laposta relation list.

== Installation ==

1. Upload `laposta-elementor-forms.php` to the `/wp-content/plugins/` directory or use wordpress plugin installer
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the Elementor editor and add a form widget to your page
4. In the form widget settings, go to the actions after submit section and add the Laposta action
